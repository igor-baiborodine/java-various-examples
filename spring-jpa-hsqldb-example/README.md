spring-jpa-hsqldb-example
-------------------------

#####This is an example of using Spring, JPA 2 with Hibernate, and HSQLDB.#####

######Software and technologies used:######
* Java
* Spring
* JPA/Hibernate
* HSQLDB
* Maven
* JUnit
* Guava
* IntelliJ IDEA
